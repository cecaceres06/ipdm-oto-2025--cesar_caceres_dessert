## Contenido implementado

- Proyecto Android con Kotlin y Jetpack Compose.
- App `Dessert Clicker`.
- Registro de callbacks del ciclo de vida de la actividad en `MainActivity`:
  - `onCreate()`
  - `onStart()`
  - `onResume()`
  - `onPause()`
  - `onStop()`
  - `onRestart()`
  - `onDestroy()`
- Estado de pantalla con `rememberSaveable` para conservar datos durante cambios de configuración.
- Interfaz para vender postres, acumular ingresos y cambiar el postre mostrado según la cantidad vendida.
